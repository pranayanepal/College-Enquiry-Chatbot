Bachelor of Computer Applications (BCA)
Overview
The Bachelor of Computer Applications (BCA) is an undergraduate degree program focused on computer science, software development, and information technology.
The program aims to develop problem-solving skills, programming expertise, and practical IT knowledge required in the modern technology industry.

Duration:-
4 Years
8 Semesters

Eligibility Criteria:-
Completion of +2 (10+2) in Management or Science
Basic knowledge of Mathematics is preferred
Eligibility criteria may vary by universit
Core BCA Subjects
The BCA program covers a wide range of foundational and advanced IT subjects:
Programming Languages
C, C++, Java, Python
Data Structures & Algorithms
Organizing, storing, and processing data efficiently
Database Management Systems (DBMS)
Database models, SQL, normalization, and database design
Operating Systems
Concepts of Windows, Linux, process management, and memory management
Web Technologies
HTML, CSS, JavaScript, PHP, and web application development
Computer Networks & Cybersecurity
Networking fundamentals, protocols, network security concepts
Software Engineering
Software development life cycle (SDLC), design models, and testing
Mathematics & Statistics
Calculus, probability, statistics, and discrete mathematics
Computer Fundamentals & Applications
Computer hardware, software, and common application packages
